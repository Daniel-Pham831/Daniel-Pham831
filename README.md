- 🌐 &nbsp;Building **Magic Monster**, a pet-collector × idle RPG, on a real-time multiplayer backend I wrote in **SpacetimeDB / Rust**
- 🍜 &nbsp;Solo-developing **Strixel**, an idle tycoon set on an old Saigon food street
